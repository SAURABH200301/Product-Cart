# Product Cart

Product Cart is an e-commerce website with a seller-side interface.  
A seller can add categories and their products in it.

## Operations that are allowed
- CRUD for products
- CRUD for categories
- User signin and signup
- Can download Excel sheet of the products
- Filters and sorting features
- Image of product is also added using Docker container

---
